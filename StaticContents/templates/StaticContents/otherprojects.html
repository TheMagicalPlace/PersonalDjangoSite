Other Projects

This page contains a brief description of and a github link to some of the other projects I have worked or been working on. Note that most of the projects shown here are either fairly minor or not fully complete, with clarification given for the latter within the project description as applicable.

Chemical Engineering Estimate Utilities

Github Link

A collection of programs for quick initial estimates for unit operation equipment design. Currently includes modules for graphical analysis of trayed distillation columns using the McCabe-Thiele method, numerical analysis of distillation towers using the FUG Method, and graphical analysis of stages required for liquid-liquid extraction using the Hunter-Nash method. For chemical reactor design, modules for batch and CSTR reactor design and analysis are partially complete, with PFR/PBR reactor functions to be implemented at a later time.

Procedural Generation One-offs

Github Link

A collection of files for testing out approaches to procedural generation. All files in the repositiory are independent of one another, with the only dependencies being the third-party modules specific to each one, with the most frequent being NumPy and matplotlib.
